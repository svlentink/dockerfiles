## Function

```javascript
lc(inp,x) // last x char of input, default x=1 (recursive function, third param is pointer)
fc(inp,x) // first x char of input
up(inp) // uppercase
lo(inp) // lowercase
rb(inp,{x:y}) // replace all x by y, default {e:3,l:7,o:0}
tr(inp,x) // truncate, default x=16
```
